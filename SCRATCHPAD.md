### Estrutura do Advertorial ("Cavalo de Troia")

```
1. HOOK (Empatia)
   → Descreva a dor do usuário melhor do que ele mesmo
   Ex: "Você abre o computador, olha pro trabalho... e congela."

2. AGITATION (Ciência do Problema)
   → Explique POR QUE acontece (Dopamina, falta de rituais)
   Ex: "Isso tem nome: Paralisia ADHD. Seu cérebro busca dopamina..."

3. AH-HA MOMENT (O Conceito)
   → Apresente a solução conceitual
   Ex: "A solução não é força de vontade. É Condicionamento Pavloviano."

4. BRIDGE (Implementação)
   → "Ok, mas como faço isso na prática sem complicação?"
   Ex: "Na prática: dedique uma playlist SÓ pra trabalho..."

5. SOFT CTA (A Ferramenta)
   → Apresente o app como a maneira mais fácil de aplicar o conceito
   Ex: "Foi por isso que criei o GettingShitDone.pro..."
```

**Regras do Advertorial:**

| ✅ Fazer | ❌ Não Fazer |
|----------|-------------|
| Citações científicas | Copy-paste de GPT |
| Insights valiosos | CTAs agressivos |
| Estudos comprovados | Texto genérico |
| Venda sutil | Parecer propaganda |
| Headline promete descoberta/método | Headline promete software |

#### Comunidades (r/ADHD, r/productivity, Hacker News)

**Abordagem:**
1. NÃO postar link do app direto
2. Postar o CONTEÚDO científico/útil
3. No final: "I built a tiny tool to automate this logic for myself, if anyone wants to try, it's free at [Link]"

**Por que funciona:** Essas comunidades amam "scratch your own itch" - devs que constroem ferramentas pra resolver problemas próprios.

**Posicionamento:** De "dono do app vendendo" → "dev com ADHD que construiu pra si mesmo"

## Decisões Tomadas

1. **Idiomas:** Ambos (PT-BR + EN) - dobra alcance, menos competição em PT-BR
2. **Plataforma:** Blog próprio (gettingshitdone.pro/blog) - controle total + SEO no domínio
3. **Sistema:** Perene e quase automatizado - conteúdo evergreen que rankeia sozinho

### Sistema Perene - Conceito

O objetivo é criar um "asset" de conteúdo que:
- Uma vez publicado, continua gerando tráfego orgânico
- Não depende de postagem constante (não é rede social)
- SEO faz o trabalho pesado depois que o conteúdo está no ar
- Cada artigo é uma "máquina de tráfego" independente
